# 🌸 Sree Lalitha Events Website

A traditional-themed, mobile-friendly event services website featuring a 3D rotating carousel and interactive contact options.

---

## ✨ Features

- 🎡 **3D Card Carousel**
  - Smooth rotating animation
  - Displays services like Bridal, Groom, Mehendi, Food, etc.

- 📞 **Smart Contact System**
  - Floating **Contact** button (top-right)
  - Tap → Shows:
    - Call
    - WhatsApp
    - Instagram
  - Tap **Call** → Shows **2 phone numbers**

- 🎨 **Traditional UI Design**
  - Golden theme
  - Elegant typography
  - Background image with overlay

- 📱 **Mobile Responsive**
  - Optimized for smartphones
  - Smooth touch experience
